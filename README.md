# Football Analytics Dashboard

A multi-screen Streamlit dashboard for football data analysis.

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Screens
1. **Prematch** — Filter & analyze historical match data
2. **Live** — Reconstruct live match state from history
3. **Testa a Testa** — Head-to-Head analysis
4. **Pattern Explorer** — Advanced filter combinations
5. **Filtri Salvati** — Manage saved filters
6. **Upload Dati** — Import CSV/Excel match data
